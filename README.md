
# Template for Node.JS API developed for my practices (I'm still working)

DB migrations and ORM:
    - Sequelize

Validation request body:
    - Join

Tests 
    - Jest (Working)

and 
    - Typescript

# Running 

1 - npm install
2 - npx sequelize db:migrate
3 - npx sequelize db:seed:all
4 - npm start
